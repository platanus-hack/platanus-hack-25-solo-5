# Mockup Photos

Este directorio contiene las imágenes de ejemplo para el mockup del iPhone en el Hero.

## Imagen Requerida

**Nombre del archivo**: `fitness-example.jpg`

**Especificaciones**:
- Dimensiones: 400x500px (ratio 2:2.5)
- Formato: JPG o WebP optimizado
- Tamaño: < 100KB
- Contenido: Foto de ejemplo de fitness/body scan (frontal, neutral, sin rostro identificable para privacidad)

## Sugerencias para la Imagen

1. **Pose**: Frontal, brazos a los lados, postura neutral
2. **Fondo**: Pared lisa (blanco/gris)
3. **Iluminación**: Uniforme y clara
4. **Privacidad**: Sin rostros visibles o usar foto stock/AI-generated

## Fuentes de Imágenes

- Stock photos: Unsplash, Pexels (buscar "fitness progress photo", "body transformation")
- AI-generated: Stable Diffusion, Midjourney con prompt de fitness progress photo
- Propia: Foto de ejemplo anónima de cliente (con permiso)

## Alternativa Temporal

Si no tienes una imagen lista, puedes usar un placeholder temporal de cualquier imagen de ejemplo hasta que consigas la definitiva.
